Reddit CLI - GitHub

A Python script I wrote to browse Reddit through the command line. Perfect for wasting time while making it look like you're hard at work. It uses urllib3, and Python's standard JSON and text wrapping libraries. With this script, one can continually browse subreddits and posts.

In the process of writing the program, I had problems accounting for differing reply depth on different comments. I fixed the problems with a recursive approach. It wasn't anything very advanced, but it felt nice to finally apply that concept in a non academic setting.
